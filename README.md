# Battleborn Mobile Scheduling APP

## Setup & Installtion

Make sure you have this version of Python "python-3.9.15" installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
Python3 app.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
